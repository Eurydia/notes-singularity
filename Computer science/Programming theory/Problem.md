# Problem

A **problem** is a binary relation $P\subseteq A\times A$ where $A$ is an arbitrary [[State space|state space]].

Consider a state space $K:=(a:\mathbb{L},b:\mathbb{L})$, where the set $\mathbb{L}$ represents the Boolean set $\{0,1\}$, one possible problem $P$ on state space $K$ is

$$
\begin{align*}
P:=&\{\\
&&\bigg(\{a:0,b:0\},\{a:0,b:0\}\bigg)\\
&&\bigg(\{a:1,b:1\},\{a:1,b:1\}\bigg)\\
&\}
\end{align*}
$$
